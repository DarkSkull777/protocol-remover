# protocol-remover
remover all protocol HTTP or HTTPS from list

```
usage: protocolremover.py [-h] -p PROTOCOL -l LIST [-o OUTPUT]

options:
  -h, --help            show this help message and exit
  -p PROTOCOL, --protocol PROTOCOL
                        Protocols to remove (comma-separated, e.g., 'http,https')
  -l LIST, --list LIST  File containing list of domains
  -o OUTPUT, --output OUTPUT
                        File to save the results (default: stdout)```
